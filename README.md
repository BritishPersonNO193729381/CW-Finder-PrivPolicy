# Privacy Policy

## 1. Overview

This Privacy Policy explains how data is collected, used, and stored when using this Discord bot and its related services (including external authentication systems such as Meta login).

By using the bot, you agree to the collection and use of information in accordance with this policy.

---

## 2. Data We Collect

### 2.1 Discord Data

When you use the bot, the following data may be collected and stored:

* Discord User ID
* Username / Display Name
* Server (Guild) ID
* Messages or content submitted via commands (e.g. world submissions, comments, tags)

This data is required for the bot to function properly.

---

### 2.2 Application Data

Depending on features used, the bot may store:

* Community world entries (title, description, codes, etc.)
* Tags, comments, ratings, and interactions
* User-related data such as balances, themes, or rewards

---

### 2.3 External Authentication (Meta / OAuth)

If you choose to link your account using Meta authentication:

* You will be redirected to an official Meta login page
* After authentication, a temporary authorization code is sent back to the server
* This code may be exchanged for basic account information (such as a unique user ID)

We do **not** store your password or login credentials.

---

## 3. How Your Data Is Used

Your data is used strictly for:

* Providing bot functionality
* Storing and displaying user-generated content
* Linking external accounts (if you choose to do so)
* Moderation and abuse prevention
* Improving features and system performance

---

## 4. Data Storage & Security

* Data is stored in a secure database system
* Reasonable measures are taken to prevent unauthorized access
* Only necessary data is stored — no excessive or irrelevant information is collected

However, no system is 100% secure, and use of the bot is at your own risk.

---

## 5. Data Sharing

We do **not** sell, trade, or share your personal data with third parties.

Data may only be shared if required:

* To comply with legal obligations
* To prevent abuse or malicious activity

---

## 6. Data Retention

Your data is stored only for as long as necessary to provide services.

Data may remain in the system until:

* You request its removal
* It is no longer required for functionality
